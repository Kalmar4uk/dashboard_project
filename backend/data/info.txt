Файлы необходимо загружать в базу в таком порядке:
1. users
2. team
3. employee
4. skills
5. employeeskills
6. min_score

users.csv - Файл для прогрузки юзеров (сотрудников)
team.csv - Файл для прогрузки команд
employee.csv - Файл для связывания юзеров с командой
skills.csv - Файл для прогрузки навыков
employeeskills.csv - Файл для прогрузки навыков сотрудников
min_score.csv - Файл для прогрузки минимального значения навыка по грейдам и должностям
